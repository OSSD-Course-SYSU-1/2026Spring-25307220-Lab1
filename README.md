# 开源项目贡献计划与本学期学习目标
## 一、开源项目贡献想法
作为开源初学者，我计划从基础入手参与开源：
1. 先参与文档类贡献，比如校对、补充项目说明，熟悉开源社区规则；
2. 后续尝试修复简单的代码bug，从入门级任务积累经验；
3. 积极测试开源项目，发现问题及时提交issue，为项目稳定性出力。

## 二、本学期课程学习目标
1. 熟练掌握Git的核心操作，包括提交、分支、推送、拉取等，能独立完成版本管理；
2. 理解GitHub的协作流程，会用fork、issue、PR等功能完成团队协作；
3. 把课堂知识和开源实践结合，提升代码编写和问题解决能力；
4. 培养开源协作思维，学会和社区开发者沟通、配合。
